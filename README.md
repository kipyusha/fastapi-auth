Система контроля доступа, где права пользователей определяются уровнем их подписки: Basic (Level 1), Pro (Level 2), Premium (Level 3). Каждый уровень предоставляет определенный набор возможностей и ограничений.

К примеру взята тема с подпиской на обучение.

Basic (Level 1) - дает доступ только к текстовой информации.
Pro (Level 2) - Имеет доступ к Basic и еще дополняется обучающимися видео
Premium (Level 3) - Содержит в себе Basic, Pro и дополняется тренажерами по темам

Пользователь с подпиской Basic (Level 1) не сможет получить доступ к Pro (Level 2) и Premium (Level 3).

Выдавать подписку может только пользователь с ролью Администратора.
